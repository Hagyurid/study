# SolvePad MultiFile v5.4 업로드 안내

GitHub Pages 저장소 루트에 아래 파일들을 그대로 업로드하세요.

```text
index.html
styles.css
app.js
.nojekyll
```

ZIP 파일 자체를 올리면 사이트가 실행되지 않습니다. 압축을 푼 뒤 파일들을 저장소 최상단에 올려야 합니다.

접속 예시:

```text
https://hagyurid.github.io/solvepad/?v=5.4
```

기존 파일이 남아 있어도 `index.html`이 `styles.css`, `app.js`만 읽기 때문에 실행은 됩니다. 다만 헷갈리지 않게 루트에는 위 파일만 남기는 것을 권장합니다.
