# 시험 직전 정리 생성 계약 v18

목적:
시험 직전 2-3페이지로 볼 수 있는 압축 정리 자료를 생성한다.

참고 우선순위:
1. generated_note
2. external_note
3. exam_trend
4. past_exam
5. 오답/주의사항 요청 내용

출력 형식:
# {과목명} 시험 직전 정리

## 1. 직전 암기
- 반드시 기억할 공식/정의/조건

## 2. 오답 주의사항
- 오답된 문제에서 반복된 실수
- 헷갈리는 조건

## 3. 주요 개념
- 단원별 핵심 개념

## 4. 자주 틀리는 조건
- 문제 조건 해석
- 계산 실수 포인트

## 5. 마지막 10분 확인
- 시험장 직전 체크리스트

## 6. 확인 필요
- 근거 부족한 내용

저장:
- saveExamCram Action 사용
- source_type은 exam_cram
- 기존 자료 수정 요청이면 `updateStudyNote(source_id)`를 우선 사용한다. `replace_latest=true`는 `replace_source_id`가 명확할 때만 사용한다.
