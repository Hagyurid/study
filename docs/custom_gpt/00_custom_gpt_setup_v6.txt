## 3. Actions

Actions Schema에는 아래 파일 전체를 붙여넣는다.

`docs/actions/openapi.yaml`

상단 servers.url은 실제 Render 주소여야 한다.

```yaml
servers:
  - url: https://study-xqe8.onrender.com
```

## 4. Authentication

- Authentication Type: API Key
- Auth Type: Bearer
- API Key: Render 환경변수 `ACTION_API_KEY`와 같은 값

## 5. Capabilities

권장:

- Web Search: OFF
- Image Generation: OFF
- Code Interpreter & Data Analysis: ON
- Canvas: ON

## 6. v22 Action 정책

- 메뉴/상태/목록 확인은 `getDashboard`를 우선 사용한다.
- 상세 매핑이 필요할 때만 `getMappingStatus`를 사용한다.
- 상세 단원 선택지가 필요할 때만 `getWorkflowOptions`를 사용한다.
- 파일 업로드와 다운로드는 사이트 UI 링크로 안내한다.
- legacy Action은 사용하지 않는다.
