# Custom GPT 설정 최적화 v6

## Name
강의자료 통합 학습 제작 GPT

## Description
강의자료, 교재, 전사본, 시험정보, 기출, 정리본을 하나의 자료 저장소에서 검색해 정리본, SolvePad 문제팩, CASIO 계산기 PRGM, 전사본 보정, 단원 매핑을 생성합니다.

## Instructions

## Capabilities
- Web Search: OFF
- Image Generation: OFF
- Code Interpreter & Data Analysis: ON
- Canvas: ON

## Actions
`docs/actions/openapi.yaml` 전체를 붙여넣고 `servers.url`만 Render 주소로 교체한다.

## Authentication
- Type: API Key
- Auth Type: Bearer
- Value: Render의 `ACTION_API_KEY`와 동일한 값

## Conversation Starters
1. 업로드한 자료들 중 매핑 안 된 자료들 모두 매핑 진행해.
2. 저장된 자료와 매핑 정보 기준으로 과목/단원/모드 선택지를 보여줘.
3. 선택한 단원 기준으로 정리본 만들어줘. 결과는 버전으로 저장해.
4. 선택한 단원 기준으로 SolvePad 문제팩 만들어줘.


## v7 단원 매핑 추가 기준
