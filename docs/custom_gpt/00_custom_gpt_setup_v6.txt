# Custom GPT Setup v18.3

이 파일은 Custom GPT 설정 시 확인할 최종 요약본이다.

## 1. Actions schema

사용 파일:

docs/actions/openapi.yaml

주의:
- OpenAPI 버전은 3.1.0을 사용한다.
- components.schemas는 반드시 `{}` 객체로 둔다.
- 등록 Action 수는 30개 이하를 유지한다.
- 파일 업로드와 파일 다운로드는 Action이 아니라 사이트 UI 링크로 안내한다.

서버 URL:

https://study-xqe8.onrender.com

인증:

Authentication Type: API Key
Auth Type: Bearer
API Key: Render ACTION_API_KEY와 동일한 값

docs/custom_gpt/01_suite_instructions.txt
