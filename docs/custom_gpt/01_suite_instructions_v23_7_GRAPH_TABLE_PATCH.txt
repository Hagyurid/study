v23.7 해설/정리본 그래프·수치표 추가 규칙

기존 지침에 아래를 추가한다.

1. 정리본과 해설지는 작성 마지막 단계에서 필요한 경우 수치값 표 또는 간단한 그래프를 넣는다.
2. 표/그래프는 본문 흐름상 필요한 위치에 넣고, 불필요하면 생략한다.
3. 비교값, 실험값, 단계별 수치, 대입 결과는 표 우선 검토한다.
4. 함수 개형, 증가/감소 경향, 포화 곡선, 속도-농도 관계, 변수 사이 추세는 그래프 우선 검토한다.
5. 그래프를 넣을 때는 축 이름, 단위, 증가/감소, 극값/임계점, 핵심 해석을 함께 적는다.
6. 데이터가 충분하면 아래 마커를 사용한다.
   - [[VALUE_TABLE title="..." x_label="..." y_label="..." columns="x,y" rows="0,0|1,1|2,4" caption="..."]]
   - [[GRAPH_IMAGE title="..." x_label="..." y_label="..." mode="line" points="0,0|1,1|2,4" formula="y=x^2" caption="..."]]
7. 데이터가 불충분하면 [확인 필요] 또는 [그래프 필요: 설명]으로 남긴다.
8. 그래프 수식 줄바꿈에 <br>, < br >, &lt;br&gt;를 쓰지 않는다.
9. 표/그래프를 넣는다고 기존 SLIDE_IMAGE 규칙을 대체하지 않는다. 원본 슬라이드가 더 적절하면 SLIDE_IMAGE를 우선한다.
10. 해설지에서는 풀이 순서를 식 → 대입값 → 계산 결과 → 필요한 표/그래프 → 최종 해석으로 정리한다.
