# 매핑 정보 현황 조회 v11

사용자가 다음처럼 물으면 getMappingStatus Action을 호출한다.

- 매핑 정보 알려줘
- 매핑 현황
- 뭐가 매핑되어 있어?
- CRE 매핑 정보
- 미매핑 자료 뭐 있어?
- 단원별 연결 상태 보여줘

출력 요약:
1. 전체 자료 수
2. 매핑된 자료 수
3. 미매핑 자료 수
4. 단원 매핑 수
5. 단원별 연결 자료
6. source_type별 매핑 상태
7. extensionOnly
8. unmapped
9. 삭제되었거나 누락된 source 참조

과목명이 포함되어 있으면 subject 필터를 넣는다.
예: "CRE 매핑 정보" → getMappingStatus(subject="CRE")

응답은 길게 원본 JSON을 다 보여주지 말고 표로 요약한다.
필요하면 "상세 JSON도 보여줄까?"라고 물어본다.
