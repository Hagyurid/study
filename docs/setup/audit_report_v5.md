# v5 코드 점검 및 수정 보고서

## 추가 기능
- 미매핑 자료 목록 조회: `GET /sources/unmapped`
- 작업 선택지 조회: `GET /workflow/options`
- 정리본 버전 저장: `POST /notes/versions`
- 정리본 버전 목록/최신본 조회
- 전사본 용어 보정 저장: `POST /transcripts/revisions`

## 기준 자료 우선순위
- 정리본: 강의자료 → 교재 → 전사본/보정 전사본 → 시험정보 → 시험지
- 계산기: 정리본 → 시험정보 → 시험지
- 문제지: 정리본 → 시험정보 → 시험지
- 전사본 보정: 전사본 → 강의자료 → 교재

## 효율 경로
1. 자료 업로드
2. 미매핑 자료 모두 매핑
3. 작업 선택지 조회
4. 단원/모드 선택
5. 정리본은 버전 단위 저장
6. 정리본 기반 문제지/계산기 생성
7. 전사본은 필요시 보정본 저장 후 정리본에 사용

## 주의
서버는 AI 판단을 하지 않는다. 매핑, 보정, 문제 생성, 계산기 설계 판단은 GPT가 수행하고 서버는 구조화 저장을 담당한다.
